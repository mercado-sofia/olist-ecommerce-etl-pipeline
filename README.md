# Olist E-Commerce ETL Pipeline

This project builds an ETL pipeline using the [Olist Brazilian E-Commerce](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce) public dataset. The pipeline extracts multiple raw CSV files, cleans and transforms the data using Python and Pandas, validates relationships across tables, loads the modeled data into PostgreSQL, and creates SQL views and queries for reporting.

## Tools Used

- Python
- Pandas
- PostgreSQL
- SQLAlchemy
- psycopg2
- python-dotenv
- SQL
- pgAdmin
- Git and GitHub

## Dataset

Dataset: Brazilian E-Commerce Public Dataset by Olist  
Source: Kaggle

The dataset contains real public e-commerce order data from Brazil, including customers, sellers, products, orders, order items, payments, and reviews.
